Document Verification
Document Verification is a web application that allows users to verify documents online. The application is built using React.js, Bootstrap, React, and Antd.

Installation
To run the application locally, please follow these steps:

Clone the repository using git clone https://github.com/Kunal-Budhrani/document-verification-frontend.git
Navigate to the project directory using cd document-verification-frontend
Install the required dependencies using npm install
Start the application using npm start
The application will be available at http://localhost:3000.

Please note that this project depends on another repository, https://github.com/Kunal-Budhrani/document-verification-backend.git. Before running the project, you will need to set up the backend repository as well.


Usage
To use the application, users can upload a document and verify its authenticity. The application will perform a series of checks to ensure that the document is valid and has not been tampered with. If the document passes all checks, the user will receive a confirmation message.
